# Double

[![travis][status]][travis]

![repliee][repliee]

## Description

Something can be [familiar, yet foreign at once][uncanny].

**Double** allows you to use a constant before it is defined when
metaprogramming.

## Usage

```ruby
class Foo
  extend Double
  Bar # nothing raised
end
```
